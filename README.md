Portfolio Website – HTML/CSS/JS + Three.js

This is a personal portfolio website built using plain HTML, CSS, and JavaScript, with Three.js added for the interactive 3D section. The idea behind this project was to create a lightweight, fast website with a custom design instead of using any templates or UI libraries.

The site is fully responsive and runs on a custom domain.

✨ Features

Custom UI built from scratch

Three.js 3D animation/visual elements

Smooth scrolling and simple interaction effects

Fully responsive layout for mobile and desktop

Lightweight—no heavy frameworks

Hosted on a personal domain

🛠️ Tech Used

HTML5

CSS3 

JavaScript 

Three.js for 3D animations

Hosting: namecheap,github pages


🚀 Running the Site Locally

No build tools required.

Just clone the repo and open the HTML file:

git clone <your-repo-link>
cd <project>


Open index.html in any browser.
(Or use a Live Server extension if preferred.)
